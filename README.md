# iteratorRemoveIt
Read in a list of words and a word to remove from the list.   Remove all occurrences of the word from the list.  You must use an Iterator

# Sample Data
a b c a b c a     a
a b c d e f g h i j x x x x     x
1 2 3 4 5 6 a b c a b c     b


# Sample Output
[b, c, b, c]
[a, b, c, d, e, f, g, h, i, j]
[1, 2, 3, 4, 5, 6, a, c, a, c]
